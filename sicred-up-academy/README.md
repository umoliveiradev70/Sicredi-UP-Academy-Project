# Sicredi Up Academy

## Visão geral

Este projeto é uma landing page institucional e promocional desenvolvida para o projeto Sicredi Up Academy, com foco em apresentar um campeonato de E-Sports voltado para jovens talentos. A página foi criada para transmitir a proposta de união entre tecnologia, competitividade, educação financeira e cooperativismo, oferecendo uma experiência visual moderna, dinâmica e inspiradora.

A interface foi pensada para funcionar como um portal de divulgação e inscrição, destacando os campeonatos disponíveis, o processo de participação, os benefícios oferecidos e a possibilidade de entrar em contato com a comunidade.

## Objetivo do projeto

O principal objetivo deste projeto é apresentar o Sicredi Up Academy como uma iniciativa que conecta:

- o universo dos games;
- o desenvolvimento de jovens;
- a educação financeira;
- o espírito de comunidade e cooperação.

A página busca engajar visitantes, despertar interesse pelos campeonatos e incentivar a inscrição de participantes interessados em fazer parte de uma experiência única.

## Público-alvo

A proposta é voltada para:

- jovens interessados em E-Sports;
- participantes que desejam competir em campeonatos online ou temáticos;
- novos associados do Sicredi que buscam oportunidades de desenvolvimento pessoal e profissional;
- pessoas interessadas em iniciativas que combinam tecnologia, esporte e educação.

## Funcionalidades principais

A landing page conta com diversas seções e recursos interativos, entre eles:

- navegação principal com links para as áreas do site;
- hero section com mensagem de impacto e chamada para ação;
- apresentação da proposta do projeto e da importância do Sicredi Up Academy;
- cards com os campeonatos disponíveis: Valorant, League of Legends e Counter-Strike 2;
- seção explicando o passo a passo para participar;
- destaque dos benefícios da iniciativa;
- formulário de inscrição com validação de campos;
- seção de perguntas frequentes;
- footer com informações institucionais e links úteis;
- animações de scroll e efeitos visuais para enriquecer a experiência.

## Estrutura do projeto

Os arquivos principais do projeto são:

- index.html: estrutura completa da página, com todas as seções e conteúdo semântico.
- style.css: estilos visuais, layout responsivo, animações e identidade visual do projeto.
- script.js: lógica de interação, animações, validação de formulário, efeitos de partículas e comportamento dinâmico da interface.

## Estilo visual

O projeto utiliza um visual moderno e imersivo, com elementos inspirados no universo gamer. Entre os recursos visuais presentes estão:

- fundo com efeito de partículas;
- gradientes e brilho verde, alinhados à identidade do Sicredi;
- cards com destaque visual para cada jogo;
- botões com estados hover e animações;
- layout responsivo para diferentes tamanhos de tela.

## Conteúdo apresentado

A página apresenta os seguintes temas:

1. Sobre o projeto
   - contexto da iniciativa;
   - visão do Sicredi sobre desenvolvimento jovem;
   - valor do E-Sports como ferramenta de formação.

2. Campeonatos
   - destaque para três jogos diferentes;
   - apresentação das características de cada modalidade.

3. Como participar
   - passos simples para se tornar participante.

4. Benefícios
   - premiações;
   - oportunidades de networking;
   - educação financeira;
   - desenvolvimento profissional e comunitário.

5. Inscrição
   - formulário com campos básicos de cadastro;
   - validação de dados;
   - feedback visual para o usuário.

6. FAQ
   - respostas para dúvidas frequentes sobre participação, requisitos e premiações.

## Tecnologias utilizadas

Este projeto foi construído com tecnologias web básicas e sem dependências externas para sua estrutura principal:

- HTML5
- CSS3
- JavaScript puro (Vanilla JS)

Também foram utilizadas imagens locais e recursos de estilo e animação diretamente no código, sem necessidade de bibliotecas adicionais.

## Como abrir o projeto

O projeto pode ser visualizado de forma simples abrindo o arquivo index.html em um navegador. Para uma experiência mais completa, também é possível usar um servidor local simples, como o Live Server no VS Code.

### Opção 1: abrir diretamente no navegador

- localize o arquivo index.html;
- clique duas vezes para abrir no navegador padrão.

### Opção 2: usar um servidor local

- abra a pasta do projeto no VS Code;
- execute a extensão Live Server ou um servidor local equivalente;
- acesse o endereço disponibilizado pelo ambiente.

## Observações importantes

- O projeto é uma interface front-end, voltada para apresentação e interação visual.
- O formulário de inscrição foi implementado com foco em experiência de usuário e validação no navegador.
- O conteúdo foi escrito de forma institucional e promocional, alinhado ao contexto do projeto.
- Não foram incluídos dados sensíveis, segredos ou informações privadas no repositório.
